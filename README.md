# epp-swarm
Exhaustive Projection Persuit

## Depenencies

* AWS SDK

https://docs.aws.amazon.com/sdk-for-cpp/v1/developer-guide/setup-linux.html

* Open SSL

```shell script
> brew install openssl
> cd /usr/local/include 
> ln -s ../opt/openssl/include/openssl
```

* FFTW

```shell script
> brew install fftw
```
